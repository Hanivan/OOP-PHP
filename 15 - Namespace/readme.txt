Namespace => Sebuah cara untuk mengelompokkan program ke dalam sebuah package tersendiri

Biasa juga disebut Encapsulation

Karena PHP tidak mengijinkan kita untuk memiliki 2 Class dengan nama yang sama. Tetapi, tidak masalah jika bekerja sendiri

Usulan Penulisan Namespace:
 namespace Vendor\Namespace\SubNamespace;
