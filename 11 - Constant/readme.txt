Constant => Sebuah identifier untuk menyimpan nilai. Nilai-nya tidak dapat berubah, berbeda dengan variabel yang nilai/value-nya yang bisa diubah
 Cara membuat constant:
  - define() ==> Ini harus disimpan diluar class
  - const ==> Ini bisa disimpan didalam class

Magic Constant:
 - __LINE__
 - __FILE__
 - __DIR__
 - __FUNCTION__
 - __CLASS__
 - __TRAIT__
 - __METHOD__
 - __NAMESPACE__
